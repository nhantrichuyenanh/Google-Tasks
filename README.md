# Google Tasks Firefox Add-on

Firefox add-on to quickly open Google Tasks from the address bar button or keyboard shortcuts.

## Features

- Sidebar:
  - Open in Sidebar (`Ctrl + Alt + G`)
  - Address bar page action button with popup controls
- New tab:
  - Open in new tab (`Ctrl + Alt + N`)
  - Open in background tab (`Ctrl + Alt + B`)

## Load temporarily in Firefox

1. Open `about:debugging#/runtime/this-firefox`.
2. Click **Load Temporary Add-on...**.
3. Select the `manifest.json` file from this folder.
